Welcome to Less Tutorial

This is a tutorial that is made for the people though who they are new to UI Development. Less is a css preprocessor to help developer to save time in development and write bug and hassle free css codes that is easy to modify and compiled

Steps to setup Less Environment

  • Install Node js(As per your ease of development)
  • open command prompt
  • Initialize npm(npm init)
    • You have to create a package.json file in which put name version and all other details.

  • To install Less (npm install less).

    *Install this globally and you'll have access to the grunt command anywhere on your system.

    *npm install -g grunt-cli.

  • A Grunt.js file will be generated in your working directory

  • create less file (eg. style.less) write your css in less format.
  • For compilation enter grunt and press enter your css file will get generated
